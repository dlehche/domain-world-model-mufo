# 领域世界模型：MUFO

## 人体功能统一本体

该仓库曾将 MUFO 表述为面向**人体功能**的领域世界模型，用于把功能结构、状态分布、风险边界与行动条件下的状态演化，表达为可计算、可审计、可治理的知识与规则系统。

## 历史定位

该仓库重点讨论：

- 将人体功能作为统一推理对象；
- 显式安全边界；
- 可解释状态推理；
- 可追溯推理工件；
- 长期回写与回放；
- 任务—能力匹配；
- 连接人体功能协作、状态解析与人体功能世界模型的三层架构。

其历史运行定义为：

> 人体在明确安全约束下，以有界的生理、力学与认知成本，稳定且高效完成任务的能力。

这一表述属于 MUFO 早期任务中心发展阶段。当前权威仓库与完整版本历史统一维护于：
