# This repository has moved

## Current authoritative MUFO repository

> **https://github.com/dlehche/MoveTips-Unified-Functional-Ontology**

This repository is preserved as a historical stage in the development of MUFO and the Human Function World Model. It is no longer the authoritative project entry point.

The current repository contains:

- MUFO Definition 2.1;
- current English and Chinese definitions;
- V1.0, Definition 2.0, and Definition 2.1 version history;
- Zenodo DOI links and citation metadata;
- current ontology architecture and claim boundary;
- research and collaboration information.

Current authoritative definition:

> **Human function is the body's capacity to be appropriately engaged to meet internal and external demands.**

Plain-language expression:

> **Human function is the body's capacity to respond appropriately when called upon.**
