# Domain World Model: MUFO

## MoveTips Unified Functional Ontology

MUFO was described here as a domain world model for **human function**, designed to represent functional structure, state, risk boundaries, and action-conditioned evolution in a computable, auditable, and safety-governed form.

## Historical position

This repository emphasized:

- human function as a unified reasoning object;
- explicit safety boundaries;
- interpretable state inference;
- traceable reasoning artifacts;
- longitudinal write-back and replay;
- Task–Capacity Matching;
- a three-layer architecture connecting functional cooperation, state parsing, and a Human Function World Model.

Its historical operational definition was:

> The capacity to complete a task stably and efficiently under bounded physiological, biomechanical, and cognitive cost within explicit safety constraints.

This wording belongs to an earlier task-centered stage of MUFO development. The current authoritative repository and version history are maintained at:

https://github.com/dlehche/MoveTips-Unified-Functional-Ontology
